## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

```sh
 gem install rails

```sh
  cd my-folder
  git clone https://github.com/kennankole/consumer-app.git
```

### Install

```sh
  cd my-project
  gem install
```

### Usage
